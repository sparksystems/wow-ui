# 介绍

----

`WOW-UI` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要用于快速开发移动端产品

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活使用的组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- node-webkit [NW.js](http://nwjs.io) （推荐）
- [Electron](http://electron.atom.io/)
